# 🧠 Artificial Neural Network (ANN)

## 📌 Overview
This project demonstrates the implementation of an **Artificial Neural Network (ANN)** for solving a machine learning problem using deep learning techniques.

## ⚙️ Steps Performed
- Data preprocessing  
- Train-test split  
- Building and training the ANN model  
- Model evaluation  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- TensorFlow / Keras  
- Scikit-learn  

## ▶️ How to Run
1. Open `18_Artifical_Neural_Network.ipynb` in Jupyter Notebook.  
2. Run all cells to train and evaluate the ANN model.

## 🎯 Conclusion
This project demonstrates how **Artificial Neural Networks** can learn complex patterns from data and be applied to machine learning tasks.
