# 🔁 Recurrent Neural Network (RNN)

## 📌 Overview
This project demonstrates the implementation of a **Recurrent Neural Network (RNN)** for sequence or time-series data analysis using deep learning techniques.

## ⚙️ Steps Performed
- Data preprocessing  
- Preparing sequential data  
- Building and training the RNN model  
- Model evaluation and prediction  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- TensorFlow / Keras  
- Matplotlib  

## ▶️ How to Run
1. Open `20_RNN.ipynb` in Jupyter Notebook.  
2. Run all cells to train and evaluate the RNN model.

## 🎯 Conclusion
The project demonstrates how **RNN models** can capture sequential patterns in data and perform predictions on time-dependent datasets.
