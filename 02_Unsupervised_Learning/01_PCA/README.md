# 📉 Principal Component Analysis (PCA)

## 📌 Overview
This project demonstrates the use of **Principal Component Analysis (PCA)** for dimensionality reduction. PCA helps reduce the number of features while preserving the most important information in the dataset.

## ⚙️ Steps Performed
- Data preprocessing and standardization  
- Applying PCA for dimensionality reduction  
- Transforming the dataset into principal components  
- Visualizing reduced dimensions  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn  

## ▶️ How to Run
1. Open `14_PCA.ipynb` in Jupyter Notebook  
2. Run all cells to execute the PCA analysis  

## 🎯 Conclusion
PCA effectively reduces dataset dimensionality while retaining most of the variance, helping improve model efficiency and visualization.
