# 📊 Clustering Analysis

## 📌 Overview
This project demonstrates **Clustering Analysis**, an unsupervised machine learning technique used to group similar data points based on their features.

## ⚙️ Steps Performed
- Data preprocessing and scaling  
- Applying clustering algorithms  
- Determining optimal number of clusters  
- Visualizing cluster groups  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn  

## ▶️ How to Run
1. Open `15_Clustering_Analysis.ipynb` in Jupyter Notebook.  
2. Run all cells to perform clustering analysis.

## 🎯 Conclusion
The project demonstrates how clustering techniques can be used to discover hidden patterns and group similar observations in a dataset.
