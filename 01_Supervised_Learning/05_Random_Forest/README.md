# 🌲 Random Forest – Classification Project

## 📌 Overview
This project demonstrates the implementation of a **Random Forest Classifier** using Python and Scikit-learn. The notebook covers data preprocessing, model training, and performance evaluation.

## ⚙️ Steps Performed
- Data preprocessing and feature preparation  
- Train-test split  
- Training the Random Forest model  
- Model evaluation  

## 📊 Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-Score  
- Confusion Matrix  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn  

## ▶️ How to Run
1. Open `12_Random_Forest.ipynb` in Jupyter Notebook.  
2. Run all cells to reproduce the results.

## 🎯 Conclusion
This project demonstrates how **Random Forest**, an ensemble learning method, can improve classification performance by combining multiple decision trees.
