# 🚗 Multiple Linear Regression – Toyota Corolla Price Prediction

## 📌 Project Overview
This project implements **Multiple Linear Regression (MLR)** to predict the price of Toyota Corolla cars using various car attributes.

The notebook demonstrates the complete machine learning workflow including data preprocessing, model building, and performance evaluation.

---

## 📂 Dataset
- **File:** `ToyotaCorolla - MLR.csv`
- **Target Variable:** `Price`
- **Features Used:**
  - Age  
  - KM  
  - FuelType  
  - HP  
  - Automatic  
  - CC  
  - Doors  
  - Weight  
  - Quarterly_Tax  

---

## ⚙️ Steps Performed
1. Data Cleaning & Preprocessing  
2. Encoding Categorical Variables  
3. Train-Test Split  
4. Building Multiple Linear Regression Model  
5. Model Evaluation  

---

## 📊 Evaluation Metrics
- R² Score  
- MAE (Mean Absolute Error)  
- RMSE (Root Mean Squared Error)  

The model explains a significant portion of variance in car prices and demonstrates good predictive performance.

This project demonstrates the practical implementation of Multiple Linear Regression for price prediction and highlights how different vehicle attributes influence car pricing.
