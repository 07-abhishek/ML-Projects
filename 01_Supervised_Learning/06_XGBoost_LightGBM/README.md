# ⚡ XGBoost & LightGBM – Classification Project

## 📌 Overview
This project demonstrates the use of **XGBoost and LightGBM**, two powerful gradient boosting algorithms, for solving a classification problem. The notebook includes model training and performance comparison.

## ⚙️ Steps Performed
- Data preprocessing  
- Train-test split  
- Training **XGBoost** and **LightGBM** models  
- Model evaluation and comparison  

## 📊 Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-Score  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- XGBoost  
- LightGBM  

## ▶️ How to Run
1. Open `13_XGBM_&_LGBM.ipynb` in Jupyter Notebook.  
2. Run all cells to reproduce the results.

## 🎯 Conclusion
This project shows how **gradient boosting algorithms** like XGBoost and LightGBM can provide strong performance for classification tasks.
